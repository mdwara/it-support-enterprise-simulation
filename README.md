# IT Support Enterprise Simulation

> **A simulated enterprise IT environment built to develop and demonstrate practical IT Support Specialist skills in a realistic business setting.**

---

## Overview

This repository documents my work as the **IT Support Specialist** for **Summit Technology Solutions**, a fictional company with 50 employees.

The purpose of this project is to simulate the daily responsibilities of an enterprise IT Support Specialist while building practical technical skills and professional documentation.

Every lesson, lab, ticket, and project completed during this bootcamp is documented here as part of an enterprise IT portfolio.
