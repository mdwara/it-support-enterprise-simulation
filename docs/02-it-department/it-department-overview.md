# Purpose

The IT Department supports all technology systems and end users at Summit Technology Solutions. Its mission is to provide reliable, secure, and efficient IT services that enable employees to perform their work with minimal disruption.

---

# IT Support Scope

The IT department is responsible for supporting:

* 50 employees
* Windows 11 laptops and desktops
* Windows Server infrastructure
* Microsoft 365 services
* Active Directory
* Microsoft Teams
* Outlook
* OneDrive
* Network printers
* VPN connectivity
* Company Wi-Fi
* Software installation and updates

---

# IT Support Responsibilities

* Respond to help desk tickets
* Troubleshoot hardware and software issues
* Reset user passwords
* Configure new employee workstations
* Maintain hardware and software inventory
* Create and update technical documentation
* Develop and maintain knowledge base articles
* Support Microsoft 365 services
* Perform basic network troubleshooting
* Escalate complex issues when necessary

---

# Ticket Priorities

| Priority | Response Goal          | Examples                             |
| -------- | ---------------------- | ------------------------------------ |
| Critical | Immediate              | Company-wide outage, server down     |
| High     | Within 1 hour          | Executive unable to work, VPN outage |
| Medium   | Same business day      | Printer issues, Outlook problems     |
| Low      | Within 2 business days | Software requests, general questions |

---

# Standard Support Hours

Monday – Friday

8:00 AM – 5:00 PM (Eastern Time)

Emergency support is available for critical business-impacting incidents.
