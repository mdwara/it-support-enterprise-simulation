# Help Desk Activities

## HD-0002 – Printer Not Printing

**Priority:** Medium

**Department:** Human Resources

A user reported that the office printer appeared to be online but was not printing documents. Performed a structured investigation by verifying the printer status, checking the print queue, reviewing the Print Spooler service, and confirming the correct default printer.

### Root Cause

The Print Spooler service had stopped after a Windows update, preventing print jobs from being processed.

### Resolution

- Restarted the Print Spooler service.
- Cleared the print queue.
- Verified the printer was online.
- Printed a successful test page.
- Confirmed the user could print normally.

**Ticket Status:** Resolved
