# Asset Tagging Standard

**Document ID:** AST-0002

**Version:** 1.0

**Author:** Kutbul Wara

**Created:** June 29, 2026

**Classification:** Internal

---

## Purpose

This standard defines the asset naming convention used by Summit Technology Solutions to uniquely identify all IT equipment.

## Naming Convention

STS = Summit Technology Solutions

* LT = Laptop
* DT = Desktop
* SRV = Server
* PRN = Printer
* MON = Monitor
* SW = Network Switch
* AP = Wireless Access Point

## Examples

* STS-LT-001
* STS-DT-001
* STS-SRV-001
* STS-PRN-001
* STS-MON-001

## Benefits

* Simplifies inventory management.
* Improves troubleshooting.
* Supports auditing and asset tracking.
* Reduces duplicate asset identifiers.
